# 空间要素与鱼塘要素相关性分析工具

基于Python的统计分析工具，用于分析自变量和因变量之间的相关性。

## 功能特点

- 支持上传Excel格式的自变量和因变量数据
- 自动数据预处理（右偏检查与处理）
- 斯皮尔曼相关性分析
- 11种曲线回归分析（线性、对数、逆、二次、三次、功率、复合、S曲线、Logistic、增长、指数）
- 结果汇总与排名
- 拟合失败记录跟踪

## 环境要求

- Python 3.10+
- uv (推荐的包管理器)

## 安装

### 使用uv安装（推荐）

```bash
# 使用uv同步依赖
uv sync
```

### 使用pip安装

```bash
pip install -e .
```

## 使用方法

### 开发模式运行

```bash
# 使用uv运行
uv run correlation-analyzer-gui

# 或激活虚拟环境后直接运行
python -m correlation_analyzer.main
```

### 打包为可执行文件

```bash
# 安装开发依赖（包含PyInstaller）
uv sync --dev

# 运行构建脚本
python build.py

# 生成的可执行文件位于 dist/相关性分析工具.exe
```

## 数据格式要求

### Excel文件要求

1. 文件格式：`.xlsx` 或 `.xls`
2. 每个文件只包含一个Sheet
3. 第一行必须包含变量名称
4. 所有数据必须是数字类型
5. 至少包含2行数据（1行变量名 + 至少1行数据）

### 变量说明

- **自变量表格**：包含作为自变量的所有列
- **因变量表格**：包含作为因变量的所有列

## 使用流程

1. **选择自变量表格**：点击"选择自变量表格"按钮，上传Excel文件
2. **选择因变量表格**：点击"选择因变量表格"按钮，上传Excel文件
3. **选择工作空间**：点击"选择工作空间"按钮，选择结果保存位置
4. **右偏检查**：点击"右偏检查"按钮，进行数据预处理
5. **开始执行**：点击"开始执行"按钮，运行完整分析

分析完成后，工作空间将包含以下文件夹：

```
工作空间/

├── 斯皮尔曼曼相关性分析结果/
│   ├── 因变量A_相关性.xlsx

│   └── ...
├── 回归分析结果/
│   ├── 因变量A_回归.xlsx
│   ├── 因变量B_回归.xlsx
│   └── ...
├── 系数结果排名汇总/
```

**注意**：本工具采用"以因变量为中心"的分析模式，每个因变量生成一组完整的相关性、回归和汇总分析报告。

### 累加值计算

**累加值 = |斯皮尔曼相关系数| + 所有模型R²之和**

累加值用于综合评价自变量与因变量的关系强度，结合了相关性分析和模型拟合优度两个维度。

## 软件版本

当前版本: v0.3.0（因变量为中心模式，累加值包含所有R²）

## 回归模型说明

工具支持以下11种回归模型：

1. **线性**: Y = b0 + b1*t
2. **对数**: Y = b0 + b1*ln(t)
3. **逆**: Y = b0 + b1/t
4. **二次**: Y = b0 + b1*t + b2*t²
5. **三次**: Y = b0 + b1*t + b2*t² + b3*t³
6. **功率**: Y = b0 * t^b1
7. **复合**: Y = b0 * b1^t
8. **S曲线**: Y = exp(b0 + b1/t)
9. **Logistic**: Y = 1 / (1/u + b0*b1^t)
10. **增长**: Y = exp(b0 + b1*t)
11. **指数**: Y = b0 * exp(b1*t)

## 边界条件处理

- 文件读取失败：弹窗提示
- 空表格：弹窗提示
- 非数字数据：弹窗提示具体位置
- 回归模型拟合失败：记录到"拟合失败记录.xlsx"

## 注意事项

1. 确保Excel文件中的数据格式正确
2. 变量名不能为空
3. 工作空间需要有写入权限
4. 大数据集可能需要较长的处理时间
