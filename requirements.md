本项目旨在基于Python制作一个简单的，功能特化的统计分析与运算工具。

## 工具执行内容

1. **用户界面可以上传两个excel表格A和B**，两个表格分别被识别为自变量与因变量的变量池，每个excel只有一个sheet，变量的名称在excel表格的第一行，所有变量都是数字类型。

2. **软件的用户界面要求可以选择工作空间文件夹**（支持在选择时新创建），在工作空间中有以下名称的细分文件夹：数据预处理、斯皮尔曼相关性分析结果、回归分析结果、系数结果排名汇总

3. **用户点击对应开始执行按钮后**，首先需要对上传的两个表格进行（非0检查），检测每一列变量是否有小于等于0的值，如果有则需要进行偏移，如果最小值为0，则右偏值为1，如果变量最小值小于0，则右偏值为'该变量最小值的绝对值+1'以确保变量中所有数值都大于0。进行完右偏处理的两个表格在工具内分别替代原有输入的数据进行后续的分析，结果存入"数据预处理"文件夹中。

4. **读取在"数据预处理"文件夹中的预处理过的存储自变量与因变量的表格**，选取其中存储**因变量**的excel中的一个变量作为运算**因变量**，分别与另一个用于存储**自变量**的excel的表格中的所有变量进行两两的运算操作，操作大致如下：斯皮尔曼相关性分析，线性与曲线回归，并将结果进行总结统计。当上述操作完成后，选取**因变量**excel表格中的另一个变量作为运算**因变量**，重复执行上述操作流程。重复该流程直到所有**因变量**excel表格中的变量都与**自变量**excel表格中的变量分别进行过操作。

## 具体分析流程（因变量为中心模式）

前提：从存储**因变量**的excel表格中选取**因变量**a。

1. **斯皮尔曼相关性分析**：使用a变量作为**因变量**与存储**自变量**的表格中的所有变量进行两两斯皮尔曼相关性分析，获取相关性分析结果数值、P值，在工作空间文件夹中的"斯皮尔曼相关性分析结果"文件夹中创建"**因变量名称**_相关性"的excel表格中存储该**因变量**的相关性分析结果，每一列按照顺序记录：**自变量名称**、相关性数值、P值。

2. **数据筛选**：对统计表格中的数据进行筛选，将P值小于0.1的**自变量**筛选出来，存储在同表格的sheet2中，存储格式与sheet1一致。

3. **回归分析**：将sheet2中的**自变量**挑选出来与**因变量**分别进行两两回归分析，每组变量的回归分析包含11种模型，提取每次回归计算中的R方，在工作空间文件夹中的"回归分析结果"文件夹中创建"**因变量名称**_回归"excel表格对结果进行存储，将每一种模型的结果存储在一个sheet中，存储结果包括：**自变量名称**，回归计算R方，具体回归函数；在该excel表格中新建一个sheet，存储对应**自变量**下的所有函数的具体R方值用于方便用户进行汇总查询。

4. **结果汇总**：将该**因变量**的斯皮尔曼相关性分析结果表格（"**因变量名称**_相关性"）中的经过P值0.1筛选的excel表格中的各**自变量**的相关性分析值提取出来并取绝对值；将"**因变量名称**_回归"中汇总的各**自变量**的回归函数R方汇总结果提取出来；将这两个提取出来的结果进行相加，结果存储在"系数结果排名汇总"中以"**因变量名称**_汇总"为名的新创建的excel表格中，按照：**自变量名称**、累加值、各函数R方值、斯皮尔曼相关系数、斯皮尔曼相关系数_绝对值 的顺序进行存储，并按照累加值对所有存储的**自变量**对应的内容进行倒序排序。

**累加值计算公式**：累加值 = |斯皮尔曼相关系数| + 所有模型R²之和

所有R²之和 = 线性R² + 对数R² + 逆R² + 二次R² + 三次R² + 功率R² + 复合R² + S曲线R² + LogisticR² + 增长R² + 指数R²

## 回归分析模型参考

回归分析中使用到的函数包括：

1. **线性**：方程为 Y = b0 + (b1 * t) 的模型。按时间的线性函数建模的序列值。

2. **对数**：公式为 Y = b0 + (b1 * ln (t)) 的模型。

3. **逆**：方程式为 Y = b0 + (b1/t) 的模型。

4. **二次**：方程为 Y = b0 + (b1 * t) + (b2 * t**2) 的模型。二次模型可用来对"减弱"的序列或阻尼衰减的序列进行建模。

5. **三次**：由方程 Y = b0 + (b1 * t) + (b2 * t**2) + (b3 * t**3) 定义的模型。

6. **功率**：方程式为 Y = b0 * (t**b1) 或 ln(Y) = ln(b0) + (b1 * ln(t)) 的模型。

7. **复合**：方程为 Y = b0 * (b1**t) 或 ln(Y) = ln(b0) + (ln(b1) * t) 的模型。

8. **S 曲线**：公式为 Y = e**(b0 + (b1/t)) 或 ln(Y) = b0 + (b1/t) 的模型。

9. **Logistic**：公式为 Y = 1 / (1/u + (b0 * (b1**t))) or ln(1/y-1/u) = ln (b0) + (ln(b1) * t) 的模型，其中 u 是上限值。选择"逻辑"之后，请指定用在回归方程中使用的上界值。该值必须是一个大于最大因变量值的正数。

10. **增长**：方程式为 Y = e**(b0 + (b1 * t)) 或 ln(Y) = b0 + (b1 * t) 的模型。

11. **指数**：方程为 Y = b0 * (e**(b1 * t)) or ln(Y) = ln(b0) + (b1 * t) 的模型。

具体信息可以参考学习：https://www.ibm.com/docs/zh/spss-statistics/31.0.0?topic=estimation-curve-models
