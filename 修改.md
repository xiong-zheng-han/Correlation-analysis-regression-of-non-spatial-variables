# 功能修改记录

## 已完成的修改

### v0.2.0 - 因变量为中心模式 (2026-04-15)

**修改内容**：

1. **分析模式反转**
   - 原来：从存储自变量的excel表格中选取自变量a
   - 修改为：从存储因变量的excel表格中选取因变量a

2. **相关性分析**
   - 原来：使用a变量作为自变量与存储因变量的表格中的所有变量进行两两斯皮尔曼相关性分析
   - 修改为：使用a变量作为因变量与存储自变量的表格中的所有变量进行两两斯皮尔曼相关性分析

3. **文件命名**
   - 原来：创建"自变量名称_相关性"的excel表格
   - 修改为：创建"因变量名称_相关性"的excel表格

4. **列名变更**
   - 原来：记录因变量名称、相关性数值、P值
   - 修改为：记录自变量名称、相关性数值、P值

5. **数据筛选**
   - 原来：将P值小于0.1的因变量筛选出来
   - 修改为：将P值小于0.1的自变量筛选出来

6. **回归分析**
   - 原来：将sheet2中的因变量挑选出来与自变量分别进行两两回归分析
   - 修改为：将sheet2中的自变量挑选出来与因变量分别进行两两回归分析

7. **回归文件命名**
   - 原来：创建"自变量名称_回归"excel表格
   - 修改为：创建"因变量名称_回归"excel表格

8. **回归结果存储**
   - 原来：包括：因变量名称，回归计算R方，具体回归函数
   - 修改为：包括：自变量名称，回归计算R方，具体回归函数

9. **结果汇总**
   - 原来：存储在"自变量名称_汇总"的excel表格中，按照：因变量名称、累加值...
   - 修改为：存储在"因变量名称_汇总"的excel表格中，按照：自变量名称、累加值...

10. **排序依据**
    - 原来：按照累加值对所有存储的因变量对应的内容进行倒序排序
    - 修改为：按照累加值对所有存储的自变量对应的内容进行倒序排序

11. **执行流程**
    - 原来：重复该流程直到所有自变量excel表格中的变量都与因变量excel表格中的变量分别进行过操作
    - 修改为：重复该流程直到所有因变量excel表格中的变量都与自变量excel表格中的变量分别进行过操作

### v0.3.0 - 累加值计算优化 (2026-04-18)

**修改内容**：

1. **累加值计算公式**
   - 原来：累加值 = |斯皮尔曼相关系数| + 最佳R²
   - 修改为：累加值 = |斯皮尔曼相关系数| + 所有模型R²之和

2. **列名变更**
   - 原来：最佳R方
   - 修改为：R方总和

3. **影响范围**
   - `src/correlation_analyzer/core/results_aggregator.py`
   - 用户使用手册.md
   - 使用说明.txt
